在网页打印99乘法表

```html

<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<title></title>
		<style>
			span{
				display: inline-block;
				width: 80px;
			}
		</style>
	</head>
	<body>
		<script>
			
			for(let i = 1;i<=9;i++){
				for(let j =1;j<i+1;j++ ){
					let k = i*j
					document.write(`<span>${i} × ${j} = ${k}</span> `)
				}
				document.write("<br>")
			}
		</script>
	</body>
</html>
```
