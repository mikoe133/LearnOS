## for循环

- For循环和while没有本质区别,都是用来反复执行代码

- 不同点就是语法结构,for循环更加清晰

- 语法:

  - for(初始化表达式;条件表达式;更新表达式){

    语句...

    }
  
- 执行流程:

  - 执行初始化表达式,初始化变量
  - 执行条件表达式,判断循环是否执行
  - 判断结果为true,则执行循环体
  - 执行更新表达式,对初始化变量进行修改
  - 重复,直到哦判断为false为止

```js
	for(let i = 0;i<5;i++){
				console.log(i);
			}

```

> **使用let在for循环中使用的变量是局部变量,只能在for循环内部访问** 
>
> **使用var在for循环中使用的变量可以在for循环的外部访问**  

- 常用死循环:
  - while(1){}
  - for(;;){}
