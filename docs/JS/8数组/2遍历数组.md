#### 遍历数组

- 获取数组中的每一个元素

- 正向遍历

  - ```js
    arr = [0,1,2,3]
    for(let i = 0;i<arr.length,i++){
      console.log(arr[i])
    }
    ```

- 反向遍历

  - ```js
    for(let i = arr.length-1;i>=0;i--){
    console.log(arr[i])
    }
    ```



#### 练习:打印未成年人信息

```js
    class Person{
        constructor(name,age){
            this.name = name
            this.age = age
        }
    }
    const personarr = [
        new Person ("aa",122),
        new Person ("bb",33),
        new Person ("cc",12),
        new Person ("dd",66),
    ]
    for(let i = 0;i<personarr.length;i++){
        if(personarr[i].age<18){
            console.log(personarr[i]);
        }
    }
```
