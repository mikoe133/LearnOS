#### 设置定时器的方式有两种

- setTimeout()

  - 参数:
    - 回调函数
    - 间隔时间(毫秒)

- ```js
  const timer = setTimeout(()=>{
  	alert(111)
  },3000)
  ```

- 关闭定时器

  - clearTimeout()

  - ```js
    clearTimeout(timer)
    ```

- setInterval()每间隔一段时间就会循环执行

  - 参数:
    - 回调函数
    - 间隔时间(毫秒)

- ```js
  const timer = setInterval(()=>{
  	alert(111)
  },3000)
  ```

- 关闭定时器

  - clearInterval()

##### 定时器的本质,是在指定时间后将函数添加到消息队列中

##### setInterval()每间隔一段时间将函数添加到消息队列中,但如果函数执行的速度比较慢,无法确保每一次执行的间隔一样

###### 若要间隔一样:反复调用setTimeout()
