##### 递归

- 调用自身的函数

编写递归函数,有两个条件

- 基线条件 ----递归终止条件
- 递归条件 ----如何对问题拆分



#### 优点:执行性能好

#### 缺点:占用空间

```js
//求阶乘
let num = 5
function jc(num){
  return jc(num-1)*num
}
```



##### 递归练习

```js
//一对兔子出生后两个月后每个月生一对小兔子,计算第n个月的兔子数
function fib(n){
  if(n<3){
    return 1
  }
  return fib(n-1)+fib(n-2)
}
let result = fib(6)
```
