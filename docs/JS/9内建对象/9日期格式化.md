- toLocaleString()将日期转换为本地时间格式的字符串

  - 参数

    - 描述语言和国家信息的字符串

      - zh-CN 中文中国
      - zh-HK 中文香港
      - en-US 英文美国

    - 需要一个对象作为参数,对象中可以通过对象的属性来对日期的格式进行配置

      - dateStyle 日期的风格
      - timeStyle 时间的风格
        - full
        - long
        - medium
        - short
      - hour12 是否采用12小时值
        - true
        - false
      - weekday 星期的显示方式
        - long
        - short
        - narrow

      ```js
      let d = new Date()
      result = d.toLocaleString("zh-CN",{dateStyle:"short",timeStyle:"short",hour12:true)
      ```
