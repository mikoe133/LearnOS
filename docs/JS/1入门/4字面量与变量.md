字面量
	字面量所代表的含义就是字面意思
	比如：1 2 3 100 ”hello“ true null .....
	在js中所有的字面量都可以直接使用，但是使用不方便

变量
	变量可以存储字面量，并且可以随意修改
	通过变量可以对字面量进行描述
	变量中并不直接存储值，而是存储值的内存地址
	
变量的使用
​	声明变量  let 变量名
​		let a
​		let b，c，d
​		var e,f,g
​	变量赋值
​		a = 10
​		a = 'hellow'
​	声明和赋值同时使用
​	let i = 100

> 声明未赋值值为undefined
>
> 未声明未赋值会报错
>
> 不声明直接赋值会变成全局变量

变量的数据类型取决于变量值

1. var可以重复声明
2. var声明变量存在声明变量提前的问题
3. 作用域为全局作用域
