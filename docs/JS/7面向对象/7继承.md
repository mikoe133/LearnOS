### 继承

- 可以通过extends关键字完成继承
- 当一个类继承另一个类时,相当于将另一个类中的代码复制到了当前类中
- 继承发生时,被继承的类称为父类,继承的类称为子类

```js
  class Animal{
        constructor(name){
            this.name = name 
        }
        sayHello(){
            console.log("动物在叫");
        }
    }
    class Dog extends Animal{
        //在子类中,可以通过创建同名方法来重写父类方法
        sayHello(){
            console.log("wwwww");
        }
    }
    class Car extends Animal{
        //重写狗仔函数时,构造函数的第一行代码必须为super()
        constructor(name,age){
            super(name)//调用父类的构造函数
            this.age = age
        }
        sayHello(){
            super.sayHello()//引用父类方法
        }
    }
    const dog = new Dog("旺财")
    const cat = new Cat("汤姆")
    dog.sayHello()
    cat.sayHello()
```

- ## 通过继承可以在不修改一个类的情况下对其进行扩展

- OCP开闭原则

  - 程序应该对修改关闭,对扩展开放
