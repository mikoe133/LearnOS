##### 在类中可以添加特殊方法constructor,是构造函数(),构造函数可以为实例进行赋值

```js
class Person{
  name
  gender
  age
	constructor(a,b,c){
    this.name = a
    this.gender = b
    this.age = c
  }
}
const p2 = new Person("孙悟空","男",18)
```
