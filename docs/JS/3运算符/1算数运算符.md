运算符（操作符）
	运算符可以用来对一个或多个值进行运算
	算术运算时，除了字符串的加法，其他运算的操作数是非数值时，都会转换为数值然后再运算
	算术运算符：

+ - * / **幂运算 %两个数相除取余

```js
a = 10 - '5'  //10-5
a = 10 + true //10+1
a = 5  + null // 5+0
a = 6  - undefined // 6-NaN=NaN
```

当任意一个值和字符串做**加法**运算时，他会先将其他的值转换为字符串，然后在做拼串的操作

```js
a = '1' + 2   //"1"+"2"=12

```

可以通过为任意类型+一个空串的形式来将其转换为字符串，其原理和String（）函数相同

```js
a = a + ""
```
