## ++ 自增运算符

- 使用后会使得原来的变量立刻增加1	
- 自增分为前自增（++a）和后自增（a++）
- 无论是++a 还是a++都会使原变量立刻增加1
- 不同的是++a和a++所返回的值不同
  - a++ 是自增前的值，旧值
  - ++a  是自增后的值，新值

```javascript
let n = 5
let result = n++ + ++n + n    //5+7+7
console.log(result)
```

## -- 自减运算符

- 使用后会使得原来的变量立刻减少1	
- 自增分为前自减（--a）和后自减（a--）
- 无论是--a 还是a--都会使原变量立刻减少1
- 不同的是--a和a--所返回的值不同
  - a-- 是自减前的值，旧值
  - --a 是自减后的值，新值
