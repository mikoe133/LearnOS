布尔值（boolean）

布尔值只有true 和 false

let bool = true

```javascript
let bool = true
console.log(bool)
```



空值 （null）

空值用来表示空对象

```
let a = null
console.log(a)
```



未定义（undefined）

当声明一个变量没有赋值时，他的值就是undefined

值的类型只有undefined

```
let b
console.log(b)
```



符号（symbol）

用来创建唯一的标识

```js
let c = symbol()//调用symbol函数创建一个符号
console.log(c)
```
