类型转换指将一种数据类型转换为其他类型

转为字符串
	1.调用toString()方法将其他类型转换为字符串
		调用xxx的yyy方法
			-->xxx.yyy()

```js
let a = true
a = a.toString()
console.log(type0f a,a)
```

由于null和undefined中没有toString（）方法，所有对这两个调用toString（）会报错



2.调用String()函数将其他类型转换为字符串
	调用xxx函数
		-->xxx()

```js
let i = 33
		i = String(i)
		console.log(typeof i,i);
```

可以转换null和undefined
