字符串（string）
	-在js中使用单引号或者双引号表示字符串
	-转义字符\
		\"  =  "
		\\  =  \
		\t  =  制表符（tab键位）
		\n  =  换行
	-模版字符串
		-使用反单引号`来表示模版字符串
		-可以嵌入变量
		
		
let a = 'hello'
a = "这是一个\”字符串\""
a = "哈哈哈\n哈洒出"
a = `哈哈
		哈
		换行`
