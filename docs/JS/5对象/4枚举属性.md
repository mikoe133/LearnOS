## 枚举属性:指将对象中的所有属性全部获取

- for-in语句

  - 语法:

    - for(let propName in 对象){

      语句...

      }

  - For-in循环体会执行多次,有几个属性就会执行几次,每次执行,都会将一个属性名赋值给我们所定义的变量

  - ```js
    			let obj = {
      				name:"sdsd",
      				name1:"ds1ds1",
            [Symbol()]:"测试属性",
      			}
      			// console.log(obj);
      			for(let propName in obj){
      				console.log(propName,obj[propName]);
      			}
    ```

  - 不是所有的属性都可以枚举,比如符号添加的属性
