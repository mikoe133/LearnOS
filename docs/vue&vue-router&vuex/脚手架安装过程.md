1、安装node.js
2、修改npm下载地址：npm config set registry https://registry.npmmirror.com
3、安装vue-cli工具（每次新电脑只需要安装一次）
	npm i webpack -g //安装webpack
	npm i @vue/cli -g //安装vue脚手架 4.5 -2.x代码
4、使用脚手架命令快速创建项目
	vue create vuedemo
      按照提示选择相应选项
5、运行项目
	cd vuedemo
	npm run serve
6、打开浏览器，查看项目
	http://localhost:8080
