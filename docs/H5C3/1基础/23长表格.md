可以将一个表格分为三个部分：
头部：thead
主体：tbody
底部：tfoot
	th表示头部单元格

```
	<table>
			<thead>
				<tr>
					<th>日期</th>
					<th>收入</th>
					<th>支出</th>
				</tr>
			</thead>
			<tbody>
				<tr>
					<td>1</td>
					<td>100</td>
		 			<td>99</td>
				</tr>
			</tbody>
			<tfoot>
				<tr>
					<td>2</td>
					<td>100</td>
					<td>99</td>
				</tr>
			</tfoot>
		</table>
```
