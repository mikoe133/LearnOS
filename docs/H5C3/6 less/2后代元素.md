```less
.box1{
	width:xxx;
	.box2{
			width:xxx;
			.box3{
				width:xxx;
			}
		}
}
```

如果需要子元素

```css
.box1{
	>.box3{
		color:xxx;
	}
}
```

如果为box1设置一个hover

$表示外围父元素

```less
.box1{
	$:hover{
		color
	}
}
```



表示p2的属性共同p1的属性，同时对p1进行扩展属性

```less
p1{
	width:100px;
}
p2:extend(.p1){
	color:red;
}
```



直接对p1的样式进行引用

```less
.p3{
	.p1();
}
```
