```
<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8" />
		<title></title>
		<style>
			table{
				width: 50%;
				border: 1px solid skyblue;
				/* 指定边框之间的距离 */
				border-spacing: 10px;
				/* 设置边框的合并 */
				border-collapse: collapse;
				margin: 0 auto;
			}
			td{
				border: 1px solid black;
				/* 文字在td中的位置 */
				vertical-align: middle;
				text-align: center;
			}
			tr:nth-child(2n){
				background-color: coral;
			}
		</style>
	</head>
	<body>
		<table>
			<thead>
				<tr>
					<th>日期</th>
					<th>收入</th>
					<th>支出</th>
				</tr>
			</thead>
			<tbody>
				<tr>
					<td>1</td>
					<td>100</td>
					<td>99</td>
				</tr>
			</tbody>
			<tfoot>
				<tr>
					<td>2</td>
					<td>100</td>
					<td>99</td>
				</tr>
			</tfoot>
		</table>
	
	</body>
</html>

```

如果表格中没有使用tbody而是直接使用tr，那么浏览器会自动创建一个tbody并且将tr全部放到tbody中，tr不是table的子元素
